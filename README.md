# 💱 Currency Converter

A simple and responsive currency converter web app built using HTML, CSS, and JavaScript that fetches real-time exchange rates via an API.

## 🚀 Features

* Convert between multiple currencies instantly
* Real-time exchange rates using API
* Clean and responsive UI
* Easy-to-use interface

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Currency Exchange API
